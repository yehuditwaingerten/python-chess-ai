# ♟️ Python Chess - Desktop Game

A desktop chess game developed in Python, allowing a player to play against the computer.  
The project demonstrates solid game logic, correct chess rule implementation, and clean code structure.

---

## Features
- Complete chess rules and legal move handling  
- Game state management  
- Computer opponent with simple decision logic  
- Tkinter-based desktop interface  
- Readable and well-organized Python code  

---

## Run
```bash
python gui_chess.py```

## About
This project was built as a personal portfolio project to practice Python development, problem-solving, and structured code design.
